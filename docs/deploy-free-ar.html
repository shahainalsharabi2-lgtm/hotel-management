خطوات رفع مشروع الفندق مجانًا

الخطة: Angular على Cloudflare Pages + API (.NET) على Render + PostgreSQL على Neon

المتطلبات قبل البدء

  1. حساب GitHub لرفع المشروع.
  2. حساب Render لنشر الـ API.
  3. حساب Neon لقاعدة البيانات PostgreSQL.
  4. حساب Cloudflare Pages لنشر واجهة Angular.

1) رفع المشروع على GitHub

  1. أنشئ Repository جديد على GitHub.
  2. ارفع مجلد المشروع كاملًا (الذي يحتوي angular و aspnet-core).

2) إنشاء قاعدة بيانات Neon (PostgreSQL)

  1. أنشئ Project جديد في Neon.
  2. انسخ Connection String الخاص بـ PostgreSQL.
مثال Connection String: Host=...;Port=5432;Database=...;Username=...;Password=...;SSL Mode=Require;Trust Server Certificate=true

3) نشر الـ API على Render (Docker)

  1. في Render اختر: New → Web Service.
  2. اربط الـ Repo من GitHub.
  3. اختر Deploy method: Docker.
  4. حدد Root Directory: aspnet-core.
متغيرات البيئة (Environment Variables):
ASPNETCORE_ENVIRONMENT = Production
Database__Provider = PostgreSql
ConnectionStrings__Default = <Neon connection string>
App__SelfUrl = <Render API URL>
AuthServer__Authority = <Render API URL>
App__ClientUrl = <Cloudflare Pages URL>
App__CorsOrigins = <Cloudflare Pages URL>

4) تشغيل DbMigrator (مرة واحدة) لإنشاء الجداول

  1. على جهازك محليًا، افتح مشروع aspnet-core/src/Modiaf.Al.Arab.Hotel.DbMigrator.
  2. ضع نفس إعدادات PostgreSQL:
Database:Provider = PostgreSql
ConnectionStrings:Default = <Neon connection string>
  1. شغّل الأمر:
dotnet run -c Release --project aspnet-core/src/Modiaf.Al.Arab.Hotel.DbMigrator

5) نشر Angular على Cloudflare Pages

  1. Cloudflare Pages → Create project → اختر نفس Repo.
  2. إعدادات البناء:
Build command = npm ci && npm run build
Output directory = angular/dist/Hotel
  1. بعد أن تحصل على رابط API من Render، حدّث ملف الإنتاج في Angular:
angular/src/environments/environment.prod.ts:
oAuthConfig.issuer = <Render API URL>
apis.default.url = <Render API URL>
application.baseUrl = <Cloudflare Pages URL>

6) اختبار سريع بعد النشر

  1. افتح رابط Cloudflare Pages.
  2. تأكد أن الصفحات تعمل (Dashboard / Rooms / Bookings / Settings).
  3. إذا ظهر خطأ CORS: راجع متغير App__CorsOrigins في Render.
