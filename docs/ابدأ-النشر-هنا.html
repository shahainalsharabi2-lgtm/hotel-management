نشر مشروع الفندق — اتبع الأرقام فقط

لا أستطيع الدخول بحسابك — لكن هذه الصفحة تفتح الروابط الصحيحة وتقلّل الخطوات.

الخطوة 1 — Render (بدون CAPTCHA)

اضغط الزر → اختر Sign in with GitHub (لا تستخدم Create Account بالبريد)

1) فتح Render + GitHub

الخطوة 2 — قاعدة البيانات Neon

سجّل → New Project → انسخ Connection string واحفظه في ملف نصي

2) فتح Neon

الخطوة 3 — ربط المشروع في Render

بعد الدخول: New + → Blueprint → اختر مستودع hotel-management

عندما يطلب قيمًا سرية، الصق سلسلة Neon في ConnectionStrings__Default

باقي الروابط يمكن تعديلها لاحقًا.

3) Blueprints في Render

الخطوة 4 — على جهازك (مرة واحدة)

في PowerShell:

cd "aspnet-core\src\Modiaf.Al.Arab.Hotel.DbMigrator"
# عدّل appsettings.json: Provider=PostgreSql + Connection string من Neon
dotnet run -c Release

الخطوة 5 — الواجهة Cloudflare

5) فتح Cloudflare

Pages → Create → GitHub → مجلد angular → Build: npm ci && npm run build → Output: dist/Hotel

بعد كل خطوة أرسل للمساعد: «انتهيت من 1» أو «انتهيت من 2» وسيكمل معك الباقي.
